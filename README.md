## About

I'm David, a developer from Germany building local-first tools, polished React/Vite interfaces, and Linux desktop setups that feel fast, clean, and practical.

My projects usually sit where power-user workflows meet good UI: terminal screenshots for docs, privacy-minded QR generation, local iOS signing inspection, and KDE Plasma customization.

## What I Build

| Project | Why it exists | Stack |
| --- | --- | --- |
| [terminal-shot](https://github.com/david-x3d/terminal-shot) | Turns commands, ANSI output, and text into polished PNG, SVG, or HTML terminal screenshots for docs and social posts. | `TypeScript` `CLI` `Playwright` |
| [Qrawisp](https://github.com/david-x3d/qrawisp) | TUI-first QR code generator for WiFi, clipboard, URLs, vCards, calendar events, geo links, and more. | `TypeScript` `TUI` `Privacy` |
| [ios-cert-checker](https://github.com/david-x3d/ios-cert-checker) | Local inspector for `.p12` certificates and `.mobileprovision` files with compatibility checks. | `TypeScript` `Node.js` `iOS` |
| [KDE Plasma Liquid Glass Theme](https://github.com/david-x3d/kde-plasma-liquid-glass-theme) | A practical KDE Plasma rice kit inspired by glossy Liquid Glass styling. | `CSS` `KDE` `Linux` |

## Current Focus

- Shipping small developer tools that solve one workflow cleanly.
- Keeping privacy-sensitive work local by default.
- Building React and TypeScript apps with sharp UX and readable code.
- Tuning Linux/KDE setups for glassy visuals without giving up speed.
